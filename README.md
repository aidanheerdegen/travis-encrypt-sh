# travis-encrypt-sh

Pure-`bash` script to [encrypt values](https://docs.travis-ci.com/user/encryption-keys/)
for use in [Travis-CI](https://travis-ci.org) build scripts.

Requires `curl`, `sed`, and `openssl rsautl` command-line utilities.
